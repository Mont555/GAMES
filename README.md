Classic PONG remake using [raylib](https://github.com/raysan5/raylib/)

### Building instructions

Install dependencies ( `gcc`,`raylib` )


GNU/Linux:
```
git clone https://github.com/Mont555/GAMES
cd GAMES/PONG2/
gcc main.c -lraylib -o PONG2
./PONG2
```
